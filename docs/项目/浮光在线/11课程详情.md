### 视频播放器

针对客户端的课程详情页的左上角内容，我们可以显示课程的详情图片，如果课程的介绍视频，也可以优先显示视频，

当然播放视频，肯定需要对应的播放器。市面上很多：百度云、腾讯云、网易云、阿里云、又拍云、七牛云或者其他第三方。

阿里云播放器，官网：https://player.alicdn.com/aliplayer/index.html

当然接下来vue3使用的vue3-video-player视频播放组件，所以我们需要先预安装。

vue-video-player地址：https://github.com/LarchLiu/vue3-video-player

demo：https://cloudgeektech.com/vue3-video-player/

安装依赖

```bash
cd fuguangweb/
yarn add @cloudgeek/vue3-video-player --save
```

在main.js中注册全局加载播放器组件

```javascript
import { createApp } from 'vue'
import App from './App.vue'
import router from "./router/index.js"
import store from "./store/index.js"
import ElementPlus from 'element-plus'
import 'element-plus/lib/theme-chalk/index.css';

import '@cloudgeek/vue3-video-player/dist/vue3-video-player.css'

createApp(App).use(router).use(store).use(ElementPlus).use(Vue3VideoPlayer, {
  lang: 'zh-CN'
}).mount('#app')

```

from .models import Teacher


class CourseTearchModelSerializer(serializers.ModelSerializer):
    """课程老师信息"""

    class Meta:
        model = Teacher
        fields = ["id", "name", "avatar", "role", "get_role_display", "title", "signature", "brief"]


class CourseRetrieveModelSerializer(serializers.ModelSerializer):
    """课程详情的序列化器"""
    direction_name = serializers.CharField(source="direction.name")
    category_name = serializers.CharField(source="category.name")

class CourseChapter(BaseModel):
    """课程章节"""

    course = models.ForeignKey("Course", related_name='chapter_list', on_delete=models.CASCADE, db_constraint=False, verbose_name="课程名称")

    class Meta:
        db_table = "fg_course_chapter"
        verbose_name = "课程章节"
        verbose_name_plural = verbose_name

    def __str__(self):
        return "%s-第%s章-%s" % (self.course.name, self.orders, self.name)

    def text(self):
        return self.__str__()

    text.short_description = "章节名称"
    text.allow_tags = True
    text.admin_order_field = "orders"

    @property
    def get_lesson_list(self):
        """获取课时列表的自定义字段"""

        return [{
            "id":lesson.id,
            "name":lesson.name,
            "orders":lesson.orders,
            "duration":lesson.duration,
            "lesson_type":lesson.lesson_type,
            "lesson_link":lesson.lesson_link,
            "free_trail":lesson.free_trail
        } for lesson in lesson_list]
