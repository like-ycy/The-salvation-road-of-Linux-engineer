# 可变与不可变数据类型

可变类型：值改变，id不变，证明改的是原值，原值是可以被改变的

不可变类型：值改变，id也改变了，产生新的值，压根没有改变原值，证明原值是不可被修改的

**需要注意，可变类型通过方法改变数据才是修改内存中的数据，使用赋值”=”号并不是修改内存中的数据，而是开辟出一块新的空间来存放新的数据。**
