## ⚡ 学习建议

- **要懂得取舍，没必要所有的东西都啃的很透**

  ?> 这个东西是什么，用来干嘛，怎么用

- **一定要培养自我解决问题的能力**

  ?> 遇到一个问题之后一定要先给自己设置一个解决问题的时间限制

  ?> 30分钟内自己解决  百度

  ?> 30分钟到一个小时，问朋友，同学

  ?> 一个小时以上，问领导
